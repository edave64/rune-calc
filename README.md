# Rune Calculator

A helper program for the card game "Rune - Saga of Heros", that helps you keep
track of various markers.

# Usage

Run `npm run dev` to start the development server.

Run `npm run build` to build the project for production. (Files will be in the `dist` directory and
can be served with any static file server in any sub-directory)

# Toolchain

This project is built with [Vite](https://vitejs.dev/) and [Vue 3](https://vuejs.org/).

During development, it uses [oxlint](https://oxc.rs/docs/guide/usage/linter.html) to lint the code
and [oxfmt](https://oxc.rs/docs/guide/usage/formatter.html) to format the code.
